
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript created for a simple API RESTFUL Library.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Finally, you can access this API through the following URL:

```bash
http://localhost:3001
```

Additionally, swagger documentation can be found at:
```bash
http://localhost:3001/doc
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
